# Dreamsmith Productivity Bot

A gamified productivity tracking system built with Python, Discord, PostgreSQL, and Heroku.

The Dreamsmith Productivity Bot was created to help users track meaningful work over long periods of time through a persistent progression system inspired by role-playing games.

Users clock in, perform productive work, clock out, and accumulate experience based on time invested. As users progress, they unlock levels, milestones, and achievement messages designed to reinforce long-term growth and consistency.

---

## Core Philosophy

Meaningful work compounds.

The Dreamsmith Productivity Bot was built around a simple idea:

Small amounts of productive effort, repeated consistently over time, can transform a person's life.

Rather than focusing on short-term motivation, the system emphasizes long-term progression and personal development.

## Technical Highlights

### Backend Architecture

The application uses PostgreSQL for persistent storage and user management.

Stored data includes:

* User IDs
* Total Productive Hours
* Clock-In State
* Progression Level

### Database Layer

The project includes a dedicated database management layer responsible for:

* User Creation
* User Retrieval
* Data Updates
* Progress Persistence
* Administrative Operations

### Deployment

The application is designed for cloud deployment using Heroku.
